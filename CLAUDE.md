# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

This repository explores integrating coding agents into the Software Development Lifecycle (SDLC). It serves two purposes:

1. **Coding agent configurations and skills** -- reusable guidelines for enterprise adoption of agentic AI in software development.
2. **A book on the topic** -- located in the `book/` subfolder, covering how to use agentic AI throughout the SDLC.

## Repository Structure

- `book/` -- Contains the book content on using agentic AI in the SDLC.
- Root-level files include coding agent configurations, skills, and examples intended as enterprise guidelines.

## Writing Conventions

- **License**: CC0 1.0 Universal (public domain). All contributions are released under this license.
- When editing book content in `book/`, preserve the author's voice and writing style. Suggest structural or factual improvements but avoid rewriting prose unnecessarily.
- Configuration and skill files at the root level should be practical, opinionated, and ready to adapt for enterprise use.
