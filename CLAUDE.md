# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

This repository is an ebook on using agentic AI in the Software Development Lifecycle (SDLC),
built as a living handbook for engineers and tech leads.

The book is read directly on GitHub — no build tools, no hosted site. Navigate via linked READMEs.

## Repository Structure

- `README.md` -- Entry point: title, introduction, table of contents.
- `book/` -- Book content organized by part.
  - Each part has a `README.md` with description and chapter links.
  - Chapters are folders with their own `README.md`.
- `skills/` -- Companion files (skills, configurations, prompts) referenced from chapters.

## Writing Conventions

- **License**: CC0 1.0 Universal (public domain). All contributions are released under this license.
- Preserve the author's voice and writing style. Suggest structural or factual improvements but avoid rewriting prose unnecessarily.
- Companion files in `skills/` should be practical, opinionated, and ready to adapt for enterprise use.
