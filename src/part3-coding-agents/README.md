# Part 3: Coding Agents

This part covers coding agents specifically. It starts with what distinguishes an agent from a chat assistant, walks through how agents work internally, and then focuses on the practical skills of prompt engineering, context engineering, and configuring an agent for repeatable behavior.

## Chapters

7. [What Is a Coding Agent?](07-what-is-a-coding-agent.md) — Distinction from autocomplete and chat; tool use + iteration as the defining properties.
8. [Anatomy of a Coding Agent](08-anatomy-of-a-coding-agent.md) — The observe → plan → act → reflect loop; tools; memory types; orchestration.
9. [Prompt Engineering](09-prompt-engineering.md) — Writing effective task prompts; structure, constraints, examples.
10. [Context Engineering](10-context-engineering.md) — Shaping what enters the context window; the CLAUDE.md pattern; legacy codebase challenges.
11. [Defining an Agent](11-defining-an-agent.md) — System prompts, conventions files, tool permissions, skills; making behavior reproducible.
