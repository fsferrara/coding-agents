# Part 2: AI Fundamentals

You do not need to understand machine learning to use coding agents effectively, but you do need a working mental model of what these systems are and how they fail. This part covers the minimum: what a language model is, how text becomes tokens and why that matters, and the failure modes you will encounter in practice.
