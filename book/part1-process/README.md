# Part 1: Process Foundations

Before a coding agent can be useful, the work it is given must be structured well. This part covers the process fundamentals that make agent-assisted development tractable: how work flows on a Kanban board, what makes a task ready for an agent to begin, and what done means when an agent wrote the code.
